
  # Back-office Reservation Management

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## GitHub Pages (gh-pages)

  1. Run `npm run build`.
  2. Publish the content of the `build/` directory as the site root in `gh-pages`.
  3. Use URLs with forward slashes, for example `./reservas/` (not `\reservas\`).
